# 更新速查

详细说明见 `部署与更新流程.md`，这里只有命令。

## 黄区 Windows：代码更新 + 打包

```powershell
git pull origin main
.\deploy\package.ps1 -Mode incremental    # 增量包（日常，只改了代码）
.\deploy\package.ps1                      # 完整包（首次 / 依赖变了）
```

产物在 `dist\`：`ldo_modeler_*.tar.gz` + `.sha256`，两个一起拷去红区。

## 红区 Linux：日常更新

把新的 `ldo_modeler_incremental.tar.gz` 传进安装文件夹，然后：

```tcsh
cd /data/.../workarea/LDO_modeling
bash update
```

## 红区 Linux：首次部署（完整包）

```tcsh
cd /data/.../workarea/LDO_modeling
sha256sum -c ldo_modeler_full.tar.gz.sha256
mkdir -p bundle && tar xzf ldo_modeler_full.tar.gz -C bundle
bash bundle/bootstrap.sh "$PWD"
```

## 红区 Linux：启动 GUI

```tcsh
bash run_gui    # 需要显示（X11/VNC）
```
